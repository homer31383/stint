# Stint

Freelance time tracking, invoicing, and project management PWA.

## Stack
React + Vite + Supabase + PWA (installable on iOS/Android/desktop)

## Quick Start
npm install
cp .env.example .env.local
npm run dev

## Supabase
Uses same project as Axiom. All tables are stint_ prefixed.
Run supabase/migrations/001_initial_schema.sql in SQL Editor.
